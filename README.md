# 📚 Librify – Online Bookstore Website  
https://bhanvi786.github.io/Librify/

**Librify** is a modern and minimal online bookstore built with clean design, smooth interactions, and a focus on readability. Whether you're browsing gift cards or sending a message via the contact page, every part of Librify is designed to deliver a calm and engaging book browsing experience. ✨

---

## 🌟 Key Features

📖 **Multi-Page Structure**  
Explore dedicated pages like **About**, **Contact**, and a themed **Homepage**. Each page focuses on clarity and flow.

🎨 **Custom Theming**  
Enjoy a well-blended theme with smooth color transitions and readable layouts.

📬 **Integrated Contact Form**  
The contact page uses **Formspree** to collect messages and make communication easier.

🖱️ **Responsive & Interactive UI**  
Includes animations and hover effects with seamless transitions.

🎁 **Gift Card Section**  
Logo updated with gift card visuals to promote offers.

## 🗂️ File Overview

### 📄 HTML Pages
- `online.html` — Main landing page  
- `about.html` — About Librify  
- `contact.html` — Contact form
- index.html — Landing/homepage

- `about.css` — About page styles  
- `books.css` — Book listing and theme styles  
- `contact.css` — Contact form styles  

### ⚙️ JavaScript
- `about.js` — About page interactivity  
- `contact.js` — Contact form handling (Formspree)

### 🖼️ Assets
